# きずな｜熊本のベビーマッサージ教室 ランディングページ

このフォルダには、Instagramから見に来てくれたママに世界観を伝え、公式LINEへご案内するためのランディングページ（1ページのウェブサイト）が入っています。

専門知識がなくても、このREADMEの通りに進めれば更新と公開ができます。

---

## 📁 ファイル構成

```
landing-page/
├── index.html     ← ページ本体（この1ファイルだけで完結）
├── images/        ← 写真を入れるフォルダ
│   ├── photo1.jpg
│   ├── photo2.jpg
│   └── photo3.jpg
└── README.md      ← この説明書
```

---

## 📸 写真の差し替え方

### 1. 写真を準備する

- 正方形（1:1）の写真を3枚ご用意ください（スマホで撮った写真でOK）
- ファイル名を以下のように変更してください：
  - 1枚目 → `photo1.jpg`
  - 2枚目 → `photo2.jpg`
  - 3枚目 → `photo3.jpg`

### 2. 写真を入れ替える

1. このフォルダの中の `images` フォルダを開きます
2. 中にある古い `photo1.jpg` `photo2.jpg` `photo3.jpg` を削除します
3. 新しく準備した3枚の写真を `images` フォルダに入れます

これだけで写真が差し替わります。

> **豆知識**
> `index.html` を開き直すと、新しい写真で表示されます。
> 写真はなるべく小さく（1枚あたり500KB以下が目安）しておくとページが速く開きます。iPhoneで撮った写真は「画像を小さくする」アプリや、Canvaで書き出すサイズを小さくすると軽くなります。

## 💬 LINEリンク（公式LINEのURL）の変更方法

ページの中で **LINEリンクを使っている場所は4か所** あります。
すべてご自身の公式LINEのURLに置き換えてください。

### 1. `index.html` を開く

- 右クリック → 「このアプリケーションで開く」→ 「テキストエディット」 で開きます
- またはVS Codeなどのエディタをお使いの方はそちらで開いてください

### 2. 「← ここにLINEのURLを入れてください」を探す

画面の検索機能（`Command + F`）で以下の文字を検索してください：

```
← ここにLINEのURLを入れてください
```

以下の4か所が見つかります：

| 場所 | どんなボタンか |
| --- | --- |
| ヒーロー（ページ最上部） | ピンクの「公式LINEで予約する」ボタン |
| メニューセクション | 緑色の「公式LINEを友だち追加する」ボタン |
| CTAセクション | 白い「公式LINEで予約する」ボタン |
| フッター（ページ最下部） | テキストリンク「LINE」 |

### 3. `href="#"` の `#` をLINEのURLに書き換える

各コメントのすぐ下に次のような行があります：

```html
<a href="#" class="btn-line">公式LINEで予約する</a>
```

この `#` の部分を、あなたのLINEのURLに置き換えます。例：

```html
<a href="https://lin.ee/XXXXXXXX" class="btn-line">公式LINEで予約する</a>
```

※ ダブルクォーテーション `"` は消さないでください。

### 4. 保存する

`Command + S` で保存すれば完了です。

## 📷 Instagramリンクの変更方法

ページの一番下（フッター）にInstagramへのリンクがあります。

1. `index.html` の中で「← ここにInstagramのURLを入れてください」を検索
2. すぐ下にある `href="#"` の `#` を、あなたのInstagramのURL（例: `https://www.instagram.com/あなたのアカウント/`）に置き換え
3. 保存

## 📝 文章の変更方法

講師プロフィールの文や、メニューの料金を変更したいときも、`index.html` をエディタで開いて、該当する文章を書き換えて保存するだけです。

初めての方は、次のような進め方がおすすめです：

1. `index.html` を **念のためコピーしてバックアップ** を作る（ファイルを選んで `Command + D`）
2. 元のファイルを開いて、変えたい文章を書き換える
3. 保存して、`index.html` をダブルクリック→ブラウザで見て確認

> 文章の前後にある `<p>` `</p>` や `<h2>` などの `< >` で囲まれた部分は **消さないようにご注意ください**。そこは「見た目のルール」を決めている大事な部分です。

## 🌐 Vercelへの公開方法（無料でネットに載せる）

Vercel（ヴァーセル）は、無料でウェブサイトを公開できるサービスです。
ドラッグ＆ドロップだけで公開できるので、一番かんたんな方法をご紹介します。

### 所要時間: 約10分
### 料金: 無料

### 手順

#### ① Vercelアカウントを作る

1. [https://vercel.com/signup](https://vercel.com/signup) を開く
2. 「Continue with Email」か、Googleアカウント／GitHubアカウントでサインアップ
3. 確認メールが届いたらリンクをクリック

#### ② このフォルダをZIPに圧縮する

1. Finderで `landing-page` フォルダを右クリック
2. 「"landing-page"を圧縮」を選ぶ
3. `landing-page.zip` というファイルができます

※ ZIPを作らず、フォルダのままアップロードしてもOKです（次のステップでフォルダごとドラッグできます）。

#### ③ Vercelにアップロードする

1. Vercelにログインした状態で [https://vercel.com/new](https://vercel.com/new) を開く
2. 画面に「Deploy a template」や「Import」の選択肢が出てきます
3. 画面下の方に **「Deploy without Git」** または **ドラッグ＆ドロップ欄** があります
4. `landing-page` フォルダごと、その枠にドラッグ＆ドロップ
5. プロジェクト名を入力（例: `kizuna-lp`）
6. 「Deploy」ボタンをクリック

#### ④ 公開完了！

しばらく待つと、`https://kizuna-lp.vercel.app` のようなURLがもらえます。
このURLをInstagramのプロフィール欄などに貼れば、誰でも見に来られます。

## 🔄 内容を更新したいとき

公開したあと、写真や文章を変えたくなったら：

### 方法A（かんたん）: 再アップロード

1. `index.html` を編集して保存
2. Vercelのダッシュボード（[https://vercel.com/dashboard](https://vercel.com/dashboard)）を開く
3. プロジェクト名をクリック
4. 「Settings」→「General」の下の方に「Redeploy」などの項目があります
5. または新しいバージョンのフォルダをまたドラッグ＆ドロップ

### 方法B（本格的）: GitHub連携

GitHubアカウントとVercelを連携すれば、フォルダを保存するだけで自動で更新されます。
慣れてきたらこちらが便利です。（最初はA方式でOK）

## 🆘 困ったときは

- **ブラウザで開いても真っ白**
  → `index.html` を直接ダブルクリックしてみてください
- **写真が表示されない**
  → `images` フォルダに `photo1.jpg` `photo2.jpg` `photo3.jpg` という名前で入っているかご確認ください（大文字・小文字も区別されます）
- **レイアウトが崩れた**
  → 編集前のバックアップに戻してください
- **LINEリンクを押しても動かない**
  → `href="#"` の `#` をLINEのURLに変え忘れていないかご確認ください
